# Bounce Game
This is a simple paddle ball game, created with Python and Tkinter.

## Description

This is a game about a ball bouncing around a canvas and using a paddle to make sure the ball does not reach the bottom of the screen.

## Controls

- Click the left arrow key to move the paddle left.
-  Click the right arrow key to move the paddle right.


## Features

- You need to click the canvas to start the game.
- The ball has a random start position on the top of the canvas.
-  The ball changes into one of 136 colors every 2 frames.
- The ball has bouncing features so it can bounce on walls and come back.
- The paddle has controls to make it move left or right.
- If the ball hits the bottom of the canvas, the game will display a game over message and the game will freeze.
- There is a score UI at the top left corner used to show the amount of times the ball and paddle come in contact.
